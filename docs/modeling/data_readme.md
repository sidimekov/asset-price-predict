# Источник данных для forecast_minimal

- Используется синтетический временной ряд длиной 6000 точек (шаг 1 час), генерируется в `scripts/train_forecast_minimal.py`.
- Генерация: случайный блуждающий процесс с мягким трендом и сезонной синусоидой + гауссовский шум `N(0, 0.6)`.
- Начальное значение - 120, сид генератора - 7
- Тренировочная/валидационная выборки разбиваются по времени (80/20)
